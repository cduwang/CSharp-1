# Open XML SDK 2.5 for Office

- https://docs.microsoft.com/zh-cn/office/open-xml/open-xml-sdk
- https://docs.microsoft.com/zh-cn/office/open-xml/open-xml-sdk?view=openxml-2.8.1
- 适合Office 2013 及更高版本
