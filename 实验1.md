LCS（Longest Common Subsequence）最长公共子序列，它是求两个字符串最长公共子序列的问题。

## 老师源码的实验结果(有整数，字符串)

- 输入：
```
{ 34, 72, 13, 44, 25, 30, 10 };
{ 34, 13, 44, 7, 25 };
```
- 输出
```
{  34}
{- 72}
{  13}
{  44}
{+ 7}
{  25}
{- 30}
{- 10}

```

- 输入：
```
{ "abc", "def", "gh", "zwd" };
{ "abc", "2", "def", "gh", "zwd" };
```
- 输出
```
{  abc}
{+ 2}
{  def}
{  gh}
{  zwd}
```
