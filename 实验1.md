# 实验1 LCS算法的实现

LCS（Longest Common Subsequence）最长公共子序列，它是求两个字符串最长公共子序列的问题。

## 实验目的和要求
- 基于参考网站的算法，通过泛型实现任何对象的LCS

## 实验时间： 两周
    
## 实验结果

- 输入：
```
{ 34, 72, 13, 44, 25, 30, 10 };
{ 34, 13, 44, 7, 25 };
```
- 输出
```
{  34}
{- 72}
{  13}
{  44}
{+ 7}
{  25}
{- 30}
{- 10}

```

- 输入：
```
{ "abc", "def", "gh", "zwd" };
{ "abc", "2", "def", "gh", "zwd" };
```
- 输出
```
{  abc}
{+ 2}
{  def}
{  gh}
{  zwd}
```
